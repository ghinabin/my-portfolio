## Image Specifications

**Recommended dimensions:**
- Hero images (marked with 500px height): 1600x500px minimum
- Content images: 1400x800px minimum
- Mobile screenshots: 375x812px (iPhone dimensions) or similar

**Format:** JPG (for photos/screenshots) or PNG (if transparency needed)

**Quality:** High quality, optimized for web (aim for < 200KB per image if possible)

**Naming convention:** All lowercase, hyphen-separated, descriptive
